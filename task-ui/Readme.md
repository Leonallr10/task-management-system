### Frontend (task-ui)

1. **Install Dependencies**  
   Navigate to the `task-ui` folder:
   ```bash
   cd task-ui
   npm install
   ```

2. **Run the Frontend Development Server**  
   Start the development server:
   ```bash
   npm run dev
   ```
   The UI will be available at **http://localhost:5174** (or another port if 5174 is busy).
